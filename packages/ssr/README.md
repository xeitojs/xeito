## Xeito SSR

This package contains the SSR functionality for Xeito. 
It is a separate package because it has a lot of dependencies that are not needed for the client side.

To find the full documentation and more links go to the main [README file](https://github.com/aerotoad/xeito).

<hr>

## Installation

```bash
npm install @xeito/ssr
```
